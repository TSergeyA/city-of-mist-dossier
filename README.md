# city-of-mist-dossier
Цифровой лист персонажа (Досье эйдоса) для НРИ "Город Тумана" (City of Mist).
Досье эйдоса (City of Mist)
Удобный инструмент для ведения персонажа в детективной НРИ "Город Туманов".

Как пользоваться:
Откройте страницу проекта (ссылка будет в разделе Pages).

Заполняйте данные о ваших Темах (Логос, Мифос, Реликвия).

Данные сохраняются автоматически в вашем браузере (localStorage).

Важно: Чтобы перенести данные на другое устройство или сохранить копию, используйте кнопки "Сохранить" (экспорт JSON) и "Загрузить" (импорт).

Особенности:
Поддержка разных цветовых тем.

Гибкое управление ключами Силы и Слабости.
